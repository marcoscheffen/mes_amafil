# Sistema MES Amafil

Sistema MES (Manufacturing Execution System) desenvolvido para a Amafil.

## Tecnologias

- Next.js (React)
- Vite
- Outras tecnologias (a definir)

## Como rodar o projeto

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

## Estrutura do Projeto

O projeto é dividido em diferentes módulos para o controle de ordens de produção, máquinas, apontamentos e relatórios, focado em trazer agilidade e transparência para o chão de fábrica.
